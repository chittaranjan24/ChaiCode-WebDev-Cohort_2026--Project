# Quotes Gallery

A simple web application that displays quotes from a free API. Users can browse through quotes, search for quotes by a specific author, and view a random quote.

## Features

- **Display Quotes:** Shows a grid of quotes fetched from an API.
- **Pagination:** Navigate through pages of quotes.
- **Search by Author:** Filter quotes by the author's name.
- **Random Quote:** Display a random quote from the current page.
- **Loading State:** Shows a loading message while quotes are being fetched.
- **Responsive Design:** The layout adjusts to different screen sizes.

## API Used

This project uses the free quotes API from [FreeAPI](https://freeapi.app/).
- **API Endpoint:** `https://api.freeapi.app/api/v1/public/quotes`

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/quotes-gallery.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd quotes-gallery
    ```
3.  Open the `index.html` file in your web browser.
