# Random Cat Viewer

A simple web application that displays random cat images and information. This project is built with HTML, Tailwind CSS, and vanilla JavaScript. It fetches data from the [FreeAPI.app](https://freeapi.app/) service.

## Features

- Fetches and displays a random cat image.
- Shows cat's name, origin, and description.
- Displays temperament tags.
- "Next Cat" button to load a new cat.
- Loading skeleton and error states for a better user experience.
- Responsive design.

## Technologies Used

- **HTML**
- **CSS** (with Tailwind CSS)
- **JavaScript** (vanilla)

## How to Use

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/random-cat-viewer.git
    ```
2.  Open `index.html` in your web browser.

## API Used

This project uses the [Random Cat API](https://api.freeapi.app/api/v1/public/cats/cat/random) from [FreeAPI.app](https://freeapi.app/).
